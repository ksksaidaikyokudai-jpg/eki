# eki. — Station Navigator for Inbound Tourists

## 技術スタック

- HTML + Vanilla JS のみ（ビルドツールなし、フレームワークなし）
- `claude-sonnet-4-6` で画像認識（vision）
- 認証：APIキーを `localStorage` の `eki_apikey` に保存
- 訓練データ：`localStorage` の `eki_training` に保存（下記構造）
- デプロイ：GitHub `main` → Netlify 自動デプロイ
- ローカル検証：`node server.js` （スマホ接続用にLAN IPを表示）

## 訓練データの保存形式

`localStorage` キー `eki_training` に配列として保存：

```js
{
  id:          Date.now(),          // タイムスタンプID
  createdAt:   '2026-04-22T...',    // ISO8601
  imageBase64: '...',                // 長辺640pxに縮小したJPEG
  aiEstimate:  { /* AIの全レスポンス */ },
  groundTruth: { station, area } | null, // 後で入力
}
```

**制約：**
- 画像は長辺640pxに縮小してから保存（localStorage容量対策）
- 上限20件（超えたら古いものから削除）
- 旧フォーマット（`result` / `correct` フィールド）は初回起動時に自動削除
